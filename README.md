# STAT-431-Final-Project
Developing a Hierarchical model to analyze the true shooting percentage (TS%) of games of players and Franchises using Markov-Chain Monte Carlo

# Steps on Gathering the Data
1. Go to the NBA database on Kaggle through this URL: https://www.kaggle.com/datasets/wyattowalsh/basketball
2. Press the 'Download' button on the top right
3. After the files have been downloaded, drag the download to your Desktop
4. Rename the folder to 'data'
5. Place the 'data' folder inside the STAT-431-Final-Project folder
